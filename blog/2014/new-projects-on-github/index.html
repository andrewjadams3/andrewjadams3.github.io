New Projects on GitHub

Mar 30th, 2014 |

I’ve just added the source code for a couple of projects I completed for Udacity’s web development course to my Github. The first is a simple blog with some basic features like user authentication and adding posts (the templates certainly need updating though). The second is a wiki which is slightly more complex, fleshed out, and user-friendly. Both apps are Python-based and hosted on Google App Engine.

I will be featuring some of my work on the projects page of this site as I continue to develop and learn. I am working through Michael Hartl’s Ruby on Rails tutorial at the moment, hopefully I’ll be able to share my first rails app soon!
